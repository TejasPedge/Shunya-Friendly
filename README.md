## Run Locally

Clone the project

```bash
  git clone https://github.com/TejasPedge/Shunya-Friendly.git
```

Go to the project directory

```bash
  cd shunya-friendly-frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Client:** React | Redux-Toolkit | ant-d

**Server:** Node | Express

**DataBase** MongoDB
